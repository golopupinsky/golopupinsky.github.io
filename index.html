Memory Purge is small status bar application for freeing your MAC memory from cached files. It is a convenient utility to clean-up your mac after running heavy applications like games, photo/video/music editors, browsers, etc. This utility is a MUST HAVE on low memory computers (4 Gb and lower). Download and start using today absolutely for free.
